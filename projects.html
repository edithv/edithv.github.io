Modified 2048 Game

Course: CS 3110 Functional Programming
Time Duration: 4 weeks
Language: OCaml

I worked in a team of 3 to build from scratch a GUI-based 2048 Game with additional features that customized the user interface to personalize the game experience for the user. This game utilizes keyboard inputs ('w', 'a', 's', 'd') to move tiles up, left, down, and right, respectively. My group added features to the game including powerups and various color themes to extend the functionality of the 2048 game. The powerups (double, half, remove, sort, shuffle) can double the value of each tile, halve the value of each tile, remove a random tile, sort tiles from increasing to decreasing order, and shuffle the tiles, respectively.
